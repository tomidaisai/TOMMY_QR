# TOMMY QR

URLからQRコードを生成し、中央にアップロードしたイラストをピクセルアート風に重ねる静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. リンクを入力します。
3. イラスト画像をアップロードします。
4. サイズ、ドットの粗さ、イラスト領域、余白を調整します。
5. `PNG保存` で書き出します。

QRコードは高エラー訂正で生成していますが、中央に載せる画像が大きいほど読み取りにくくなります。公開前にスマートフォンで読み取り確認してください。

## 構成

- `index.html`: 画面構造
- `styles.css`: レイアウトと見た目
- `app.js`: QR生成、画像のピクセル化、PNG保存
